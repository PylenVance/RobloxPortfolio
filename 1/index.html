Project 1: DataManagement system

This project uses the DataStore service to manage player data, creating a leaderstats folder with an IntValue named Diamonds for each player upon joining, loading and saving diamonds data using DataStore on PlayerAdded and PlayerRemoving events respectively, ensuring persistence across sessions by associating data with each player's UserId.
